## 一、v2.2→v2.4 核心变更

| | v2.2（老版本） | v2.4（新版本） |
|------|-------------|-------------|
| 镜数 | 18 | 10（含序章镜0） |
| 总时长 | 74s | 74s |
| 叙事 | 现代→古代→现代（嵌套闪回） | 珍珠序章→古→泪化珠→穿越→现代（直线时序） |
| 单镜长 | 3-6s | 3-12s（镜0仅3s序章钩子） |
| 开场 | 珍珠在指间（现代） | 黑底珍珠→硬切晨雾西施浣纱 |
| 穿越段 | 单镜13(6s) | 镜4+镜5(18s，含音乐切换) |
| 爆点 | 镜10(6s) | 镜3(12s，纯白虚空+静默4s) |
| 现代女孩 | 镜14-16（故事中段插入，不明确） | 镜6-8（独立完整段落，叙事终点） |

### 变更原因

朋友反馈三项问题：
1. 穿越感太弱——镜13仅6s水彩过渡，观众感知不到2500年跨度
2. 频繁切镜导致不连贯——18镜×平均4s=切换过多，认知负荷高
3. 现代女孩不够明显——夹在中间无独立镜头，身份模糊
