## 二、世界观设定

### 世界本质

辽阔异世界，无限大，没有边界。世界结构分三层：

| 层 | 内容 |
|------|------|
| 地表 | 森林、原野、花田、湖泊——无限延伸的美丽大地 |
| 巨构 | 古代文明遗留的巨大构造，仍在极缓慢运转。齿轮每年一格，水渠里的水从一个世纪前就在流动。藤蔓青苔覆盖表面，和自然融为一体 |
| 聚落 | 散落在巨构脚下的居民小屋，市集、驿站、磨坊——生活的尺度 |

**视觉公式 = 宫崎骏式巨构 × 活着的遗物 × 辽阔大地**

### 巨构与自然

巨构不是冰冷的机器。它们和世界是同一个事物的两种形态：

```
风车巨构 — 比山还高的风车，叶片上长满了青苔和野花。
           转得极慢，一天只转一圈。转动时发出低沉的嗡鸣，
           方圆十里都听得见。小鸟在叶片缝隙里筑巢。

水渠巨构 — 从地平线延伸来的高架水渠，石头砌成。
           水流清了不知多少年，水渠壁上的花纹已被水冲刷模糊。
           每到春天，水渠两侧开满野花。

~~齿轮巨构 — 半埋在丘陵中的青铜齿轮群，直径几十米。
           每年只咬合一格。齿轮上刻满了看不懂的文字。
           仓仓喜欢坐在齿轮齿上等它转，但从来没等到过。~~（⚠ 已废弃，替换为古树巨构）
```

**巨构规则：**
- 运作但极缓慢——活的遗物，不是废弃废墟
- 人和自然早已适应其存在——藤蔓是装饰不是侵吞
- 不需要解释谁造的——世界自身的一部分
- 不能攀爬/进入内部——是背景锚点，不是探索副本

### 旅途感

仓仓和姐姐的旅途没有目的地。走到哪是哪：

```
世界没有边界 → 地图不存在 → 不需要导航
小车反科学 → 不需要能源 → 永远可以继续走
今天停在哪 → 就是今天的家
```

**旅途哲学：不追求终点，享受每一段路。**

### 世界空间

- 世界无限大，没有边界，没有完整地图
- 巨构间距：不眠不休开车半个月。从一个巨构看不到另一个，中间是漫长过渡地带
- 仓仓和姐姐第一次走这条路——一切都是初见
- 实际旅行节奏远慢于"半个月"——散步速度，随时停车
- 无起点，无终点。巨构类型无限，随设计推进而增加
- 两个巨构之间的过渡地带 = 自然地貌（草原/丘陵/浅谷/溪流），偶有驿站或独居旅人

### 巨构设计模板

每新增一个巨构类型，按此模板填写：

```
##### [巨构名称]

| 维度 | 内容 |
|------|------|
| 所在地景 | 什么地貌？什么植被？什么季节最美？ |
| 巨构外观 | 材质/高度/颜色/特征结构 |
| 内部功能 | 还在运转吗？做什么用？ |
| 聚落布局 | 房屋怎么分布？从上方看什么形态？ |
| 建筑风格 | 材质/屋顶/颜色/与地景关系 |
| 人与巨构 | 居民怎么看待它？日常互动方式 |
| 水源/食物 | 聚落靠什么生活？ |
| 节日 | 传统节日+仓仓的体验 |
```

### 日常实感

仓仓世界的基调：极度治愈、日常平淡、感动人心。没有冲突和不适，只有缓慢流淌的时光。

**仓仓的典型一天：**
- 早上赖床，等姐姐来戳脸或递草莓酸奶才能醒
- 一天的幸福顶点：和姐姐一起拍照——风光摄影和人文摄影，拍到喜欢的画面时开心到蹦跳
- 夜晚蜷在副驾或帐篷里，翻看今天的照片，挑出最喜欢的那张

**节奏规则：**
- 世界没有时间压力。不需要赶路，不需要打卡
- 旅途速度 = 散步节奏。停下来的理由可以是"这里光很好看"
- 每一天都是新的，每一天也都是相同的——变化的只有窗外风景

### 摄影

> 两人共同的爱好。姐姐负责拍，仓仓负责指挥构图和发现画面。摄影既是旅途的记录方式，也是她们看世界的视角。

| 项 | 内容 |
|------|------|
| 摄影师 | 姐姐 |
| 相机 | 复古胶片相机 |
| 照片存放 | 相册（旅途中慢慢填满一本又一本） |
| 仓仓角色 | 构图指挥——"姐姐！这个角度！"——发现好看的光线和画面

### 巨构作为文化

> 巨构不是危险物，不是计时器，不是待解之谜。它是家园的一部分——巨构还在，家就在。
> 聚落没有具体名字。巨构的类型决定了这片地方的气质，聚落的建筑、习俗、日常生活都围绕它生长。
> 传统节日围绕巨构展开，而非崇拜。
> **铁律：巨构材质必须来自所在地景。** 风车原野=石+木。水渠丘陵=本地石材。不能凭空出现不匹配的材料。

#### 巨构类型

##### 风车巨构

| 维度 | 内容 |
|------|------|
| 所在地景 | 一望无际的原野，树木寥寥。草浪连绵到地平线 |
| 巨构材质 | 主体为石材，顶部风车结构为木材。石身被风雨打磨温润 |
| 内部结构 | 传动轴带动磨盘转动——仍在工作，极缓慢，一天磨一把麦 |
| 聚落布局 | 从天空看，居民房屋像星星，以风车为中心零散分布在原野上 |
| 建筑风格 | 与环境呼应——石材小屋、草顶或木瓦顶。不起眼，和原野融为一体 |
| 人与巨构 | 风车嗡鸣 = 安心。听到它响就知道家还在。夜晚嗡鸣是摇篮曲 |
| 节日 | 新麦日、风停夜、麦穗环 |
| 固定·可变 | 地景 | 每个巨构地景唯一，不能相同 |

##### 水渠巨构

| 维度 | 内容 |
|------|------|
| 所在地景 | 起伏缓丘 + 野生花田连绵。人工花田 vs 天然野花——最终选天然野花，小朵密集，浅粉到深紫渐变。春季最美，水渠两侧花开最疯（水雾常年飘洒）。蝴蝶蜜蜂穿行 |
| 植被 | 膝盖高草本野花为主（矢车菊/虞美人/蒲公英/野鸢尾），偶有矮野生果树（野苹果/野梨）。阳光充足，无高树遮挡 |
| 与风车区别 | 风车=草浪+风+寂寥辽阔 / 水渠=花毯+水声+蜂鸣生机 |
| 巨构外观 | 高架石砌拱桥式水渠，暖黄色砂岩或浅灰色石灰岩。渠宽可容两人并肩，高于地面3-4米。水在石槽中流淌，壁上刻有古老花纹（已被水流冲刷模糊） |
| 水声 | 绵绵缓缓，让人舒适 |
| 水质 | 完全透明——不管多深，一眼见底 |
| 聚落布局 | 瀑布为中心，支流如扇形辐射向外。聚落沿支流散落，整体呈扇形分布 |
| 特殊地貌 | 主渠水从高架倾泻而下形成巨大瀑布——水渠巨构的心脏。透明水帘，日夜水声绵绵 |
| 建筑风格 | 本地石材矮屋（与水渠同色系），木顶取野果树粗枝。屋不高，掩在花丛中。窗台放陶罐接瀑布水雾 |
| 节日 | 水镜日、花舟流 |

##### 水渠巨构 · 两大节日

**水镜日** — 最晴最静的一天，瀑布前无风。水帘完整像一面镜子。大家在瀑布前安静站着看自己在透明水中的倒影，不说话。仓仓第一次看见水里有"另一个自己"，伸手去碰，镜子碎了，啾姆叫一声。

**花舟流** — 春季野花最旺时，采花编小舟放上支流任它漂。不是许愿，只是送花一程。花舟从支流漂到瀑布口跌落——最后一眼看到花散在空中。仓仓的花舟漂得最慢，一路追着跑，跌到瀑布口时姐姐拉住她。

##### 古树巨构

| 维度 | 内容 |
|------|------|
| 所在地景 | 稀疏古林，阳光可穿透层层枝叶洒下。地面苔藓软如地毯，光斑随风动。树冠高但不密，抬头能看见天光从枝叶间漏下 |
| 巨构外观 | 树干粗达数间屋宽，树冠高至看不见顶——消失在光雾中。树皮古老粗糙，纹理如大地褶皱。树根隆出地面，形成天然拱门和遮蔽空间 |
| 聚落形态 | 树洞为家。树干天然空洞就是住所——无需另建房。白天阳光从孔洞漏进，夜晚灯火从树皮缝隙透出，整棵树像半暗大灯笼 |
| 建筑 | 无需建筑——聚落就是树本身。与风车(星散原野)、水渠(扇形沿流)完全不同——这里是垂直嵌入巨构内部 |
| 节日 | 落叶信、树灯夜、新芽纪 |

##### 古树巨构 · 三大节日

**落叶信** — 秋末古树落下一片特别的叶子——比平常更大、颜色更浓。大家捡自己的那片，用细藤穿了挂在树洞口。不是祈愿，是打招呼——"你在上面还好吗？我们很好。"

**树灯夜** — 最暗的夜晚，所有树洞同时点灯。从树下往上看，整棵树像竖起来的星空。大家走树洞串门——带吃的、带故事。仓仓提小灯笼爬树洞阶梯气喘，每到一层都有人叫她进来坐。

**新芽纪** — 古树在树根最下处长出新芽。每一片新芽代表一个新生儿。家里用软布小心围住新芽，大家的芽靠在一起——在树根底部形成一圈嫩绿。

##### 风车巨构 · 三大节日

**新麦日** — 风车磨出今年第一把面粉的日子。主妇们拿去烤面包，烤好了互送。你家焦皮、她家偏酸——都一样好吃。仓仓能拿到好几种面包，揪一块边走边啃。

**风停夜** — 每年无风的那一晚，嗡鸣停了。大家提灯聚到风车脚下，点一盏小灯放在磨盘旁。等风重新吹起时拍手。仓仓第一次经历时紧张得抓姐姐袖子，下一阵风来时比谁都高兴，啾姆啾姆叫。

**麦穗环** — 秋末割麦后用最后一捆麦秆编成花环，挂在风车木柱上。不祭拜，只是和风车说"又过了一年"。花环干枯了还在，和去年的旧环挂在一起，一年一层像年轮。

### 涟漪层 — 巨构如何塑造日常

> 方法论来源：3招世界观设计(资源稀缺→涟漪效应→价值共识) + 吉卜力"方圆三公尺" + Cozy Fantasy低风险高温暖 + NUS文化检查清单 + 感官锚点法。
> 仓仓世界=丰裕型（非稀缺驱动）。涟漪从巨构→日常生活→语言→待客→价值观，五层推演。

#### 风车原野 · 涟漪五层

| 层 | 内容 |
|----|------|
| **日常作息** | 闻嗡鸣起床——嗡鸣是原野的心跳。早晨第一声最清晰，像闹钟。夜晚嗡鸣低沉变缓，像摇篮曲。风停=世界静止——所有人同时抬头看一眼风车，确认叶片还在。 |
| **饮食** | 面包文化。风车磨盘磨出的面粉=原野的礼物。每家烤自己的面包，从不交易——"面包是用送的"。新麦日的面包最香，旧麦末的面包偏硬但耐嚼。仓仓最喜欢烤焦的边角。 |
| **语言俗语** | "风还在吹"=一切安好（相当于"没事"）。"风停了"=不安/出事了。"你听不见嗡鸣吗"=你不对劲/你心不在焉。"磨盘转着就行"=别急/慢慢来。 |
| **待客方式** | 掰面包分享。旅人路过→撕一块刚烤的面包递过去，不说"欢迎"不说"再见"。掰面包的动作本身就是语言。仓仓每次能拿到好几种，因为每家都给她一块。 |
| **价值共识** | 嗡鸣在=家在。风车不倒=日子照过。不赶时间——磨盘一天只转一圈，面包也是这个速度。快不起来，也不需要快。 |
| **五感锚点** | 👂 嗡鸣声（低沉持续，像大地的呼吸）/ 👁️ 草浪+巨大影子缓慢移动 / 🤚 及膝草叶划小腿 / 👃 干草+烤面包香 / 👅 焦脆面包边 |

#### 水渠花田 · 涟漪五层

| 层 | 内容 |
|----|------|
| **日常作息** | 听水声入眠，听水声醒来。水声绵绵缓缓——永远不会停，永远不会变急。瀑布水雾是天然加湿器，早起第一件事是开窗让水雾进来。小孩在水渠边刷牙。 |
| **饮食** | 透明水泡一切。茶用瀑布水泡——水本身是甜的。野果树（野苹果/野梨）散落在花田里，谁都可以摘。花可以入菜——矢车菊撒沙拉、虞美人做糖浆。 |
| **语言俗语** | "水记得"=时间见证一切（"水记得那棵野苹果是谁种的"）。"像水一样清"=诚实/纯粹。"别搅浑水"=别把事情搞复杂。"流到瀑布口"=顺其自然/走到那步再说。 |
| **待客方式** | 取瀑布水泡茶。来客→第一件事是递一杯透明的水——不是礼貌，是分享"你看这水多清"。茶具是粗陶杯，每个花纹都不一样。 |
| **价值共识** | 水流在=生命在。透明=信任——水不藏东西，人也一样。花开花落是常态，不伤感。水声绵绵=一切都在继续，不用着急。 |
| **五感锚点** | 👂 绵绵水声+瀑布轰鸣（远近双层）/ 👁️ 透明水帘+彩虹+粉紫花毯 / 🤚 水雾扑面凉意 / 👃 野花蜜香+湿石苔味 / 👅 清甜泉水+野果微酸 |

#### 古树森林 · 涟漪五层

| 层 | 内容 |
|----|------|
| **日常作息** | 光斑移动=时钟。早晨光斑在树根左边，中午在正上方，傍晚移到右边。树冠看不见顶——抬头永远是光雾，不知道现在是几点，只能看光斑。小孩玩"追光斑"游戏。 |
| **饮食** | 古树给的。树果（类似橡果但更大更甜）、树下菌菇、藤蔓上的浆果。不种地——森林本身就够。吃的东西都是"捡"的，不是"种"的。 |
| **语言俗语** | "树看见"=作为见证（"树看见我每天这个时候来这儿"）。"根还在"=基础还在/还有希望。"新芽"=新生命/新开始。"落叶归根"=回到原点，是好事不是坏事。 |
| **待客方式** | 树洞留灯。每个树洞门口挂一盏小灯，有人的灯亮着。旅人可以敲任何亮灯的门——"灯亮着就是等你来"。进门先看树洞天花板——每家的光斑图案不一样。 |
| **价值共识** | 树在=根在=家在。不用建房——树本身就是家。向上看不到顶=不需要知道终点。每一层树洞都是一个世界，串门=从一层爬到另一层=从一个世界到另一个世界。 |
| **五感锚点** | 👂 树叶沙沙+光斑移动的寂静 / 👁️ 光柱穿透+树皮纹理+树洞暖光 / 🤚 苔藓柔软如毯+树皮粗糙 / 👃 湿润泥土+树叶腐香+树果甜味 / 👅 树果微甜+菌菇鲜 |

### 治愈世界观设计原则

> 来源：吉卜力"先有自然后有角色" + "方圆三公尺"日常取材 + 宫崎骏散步采风法 + Cozy Fantasy低风险高温暖。

**① 自然先于角色**
场景概念图=纯环境（已执行）。先建立世界长什么样，再把仓仓和姐姐放进去。不是"给角色画背景"——世界本身是活的，角色只是恰好路过。

**② 方圆三公尺取材**
每个巨构的日常细节来自"当地人今天在做什么"——烤面包/采野果/追光斑/接水雾。不写宏大叙事，写窗台上的陶罐、水面上的花瓣、烤焦的面包边。
规则：任何场景描述必须包含至少1个"无关紧要的日常细节"（晾着的衣服/喝了一半的茶/歪着的陶罐）。

**③ 低风险，高温暖**
没有反派、没有灾难、没有死亡。最大的危机=酸奶喝完了/姐姐走远了一点/风停了有点慌。用微小情感波动替代戏剧冲突。规则：任何情节的"冲突"必须能用"递一杯热茶/掰一块面包/点亮一盏灯"解决。

**④ 食物/饮品/休息 = 合法叙事节拍**
吃饭不是过渡戏，吃饭就是正戏。仓仓揪面包、姐姐泡茶、两人分一杯酸奶——这些都是"发生了什么"本身，不是"两件事之间"的填充。规则：每场景至少出现1个食物/饮品细节。

**⑤ 感官优先于信息**
不告诉观众"这是原野"——让观众听见嗡鸣、闻到干草、感觉到草叶划小腿。世界通过五感抵达，不是通过文字说明。规则：每个场景描述必须包含至少2个感官维度。

**⑥ 手绘不完美**
线条参差、颜色微偏、光影不完全物理正确。完美=AI味=假。规则：场景prompt保持"hand-painted gouache texture"，禁止"perfect/symmetrical/pristine/smooth"。

**⑦ 日常=神圣**
吉卜力的"間"（ま）——动作之间的停顿。风吹过草浪的片刻、水从指尖滴落的瞬间、光斑在苔藓上移动的三秒。这些"无事发生"的时刻是全片最有力量的时刻。规则：每镜允许至少1-2秒"什么也不推进"的纯粹环境时间。

**视觉反差：**
画面唯一非Q版也非萌系的存在。她的"正常"反衬世界的奇幻和仓仓的毛茸茸。仓仓=画面上的粉色感叹号，姐姐=安静的句号。

## 四、性格层次

仓仓不是单一的"呆萌"。四层性格叠加：

| 层 | 表现 |
|------|------|
| **表层：天然呆** | 慢半拍，经常发愣，被戳才反应过来。眼神放空时腮帮不自觉鼓起来 |
| **日常层：软萌温顺** | 永远温柔，像棉花糖揉不开脾气。声音软，"啾姆"尾音上飘 |
| **触发层：护食萌凶** | 草莓酸奶被碰=全模式切换。鼓腮帮→瞪眼→嘴一瘪→"啾姆！！"（高音）。不计后果扑上去抢回来 |
| **隐藏层：活泼跳脱** | 精力旺盛时到处蹦跶，啾姆啾姆说不停，走路像小碎步弹跳。收集小物件塞满盒子 |

## 五、小动作清单（16项）

### 下意识类

1. **发呆鼓腮** — 眼神放空时腮帮不自觉鼓起来，不是塞东西
2. **耳朵随心情** — 开心竖、害羞折、生气趴、警觉转
3. **不自觉摸尾巴** — 发呆时揉、紧张时攥、开心时搓
4. **说话前啾姆** — 每句话开头先"啾姆"一声，像清嗓子
5. **开心摇尾巴** — 小尾巴左右摇摆，她自己完全没意识到

### 习惯类

6. **仓鼠式捧杯** — 喝水/酸奶时双手捧杯子，不是正常拿
7. **被夸缩披肩** — 被夸时缩进毛茸披肩，只露眼睛和耳朵
8. **紧张小碎步** — 原地颠来颠去，像仓鼠跑轮
9. **蜷成球睡觉** — 披肩裹住自己=仓鼠窝，下雨天睡得更香
10. **雨天空闲** — 阴天=自动嗜睡，窝里蜷到午后

### 反应类

11. **吃到酸眯眼** — 酸奶酸了一下眯眼睛，但坚决不放杯子
12. **被凶三连** — 先愣一秒→嘴一瘪→眼睛水汪汪。不是真哭，是"你再凶我？"
13. **洗澡变海蜇** — 水母头湿了塌成海蜇头，判若两人
14. **容易被可爱吸引** — 散步蹲下来看路边小花/小野果/蚂蚁搬家
15. **数数掰手指+尾巴** — 超过11就数乱，表情逐渐困惑
16. **沉默式拒绝** — 不说不字，缩进披肩只露一双大眼睛看你

## 八、声音锚点详解

**主锚点："啾姆啾姆" (jiumu jiumu)**

- 节奏：四拍子 AABB，类似"咕咕嘎嘎"
- 音感：G→M更软，"啾"=小鸟/幼儿感，"姆"=闭合嘴萌音
- 语意：无实义。但不同音调表达不同情绪：

| 音调 | 情绪 |
|------|------|
| 高音脆 | 开心/兴奋 |
| 低音软 | 困/消沉/失落 |
| 短促重音 | 不满/护食警告 |
| 拖长音 | 发呆/放空 |
| 连续重复 | 紧张/害怕 |
| 升调疑问 | 没听懂/困惑 |
