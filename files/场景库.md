# 仓仓 — 场景库

> 长期计划。每隔一段时间设计一个场景。
> 每个场景 = 文档记录 + 概念图prompt（7张/场景：空间4+节日2+过渡1）。

---

## 场景模板

```markdown
### 场景 N: [名称]

**世界观锚点：**
- 巨构类型：[风车/水渠/古树/待定义]
- 自然地景：[草原/花田/森林/湖泊/山丘/峡谷/待定义]
- 时间：[晨光/午后/黄昏/夜晚/阴天/微雨/薄雾]
- 季节：[春/夏/秋/冬]

**空间关系：**
- 巨构位置：[画面中方位+距离]
- 小车位置：[相对巨构+相对人物的位置]
- 仓仓位置：[在做什么/在小车旁/在巨构脚下/在自然中]
- 姐姐位置：[开车/倚车门/坐草地上/和仓仓一起]

**旅途上下文：**
这是旅途中的哪种时刻？
- [ ] 刚停车——被景色吸引停下来
- [ ] 已安顿——正在露营/休息/做饭
- [ ] 路过——车还在动，窗外是风景
- [ ] 偶遇——发现了什么意外的惊喜
- [ ] 等待——原地停留等天气/等巨构转动

**情绪落点：** [一句话描述气氛]

**注：** 待AI生成概念图prompt。
```

## 场景清单

### 参考示例 A: 风车草原

- 巨构：风车巨构（比山高，叶片覆盖青苔野花）
- 地景：草原（草高及膝，风吹成浪）
- 时间：午后
- 空间：风车在远景，巨大叶片影子投在草原上。小车停在草地中间。仓仓在草丛里蹦跳采野花，姐姐倚车门看着
- 旅途时刻：刚停车
- 情绪：开阔自由

### 参考示例 B: 水渠花田

- 巨构：水渠巨构（高架石砌，水清了百年）
- 地景：花田（春天野花满开，水渠两侧最密集）
- 时间：晨光
- 空间：水渠从画面左侧延伸到右侧高处。花田铺满下部。小车停在水渠脚下。仓仓踮脚伸手试水渠滴落的水滴，姐姐坐在渠墩上记东西
- 旅途时刻：已安顿
- 情绪：宁静晨光

### 参考示例 C: 古树森林（已启用·替代齿轮）

- 巨构：古树巨构（树干数间屋宽，树冠消失光雾中）
- 地景：稀疏古林（阳光穿透，苔藓地毯，光斑随风）
- 时间：午后
- 情绪：神圣静谧
