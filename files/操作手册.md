## 4. GPT-SoVITS TTS 操作

### 4.1 环境

| 项目 | 值 |
|------|-----|
| Python | 3.12.9 (system) |
| torch | 2.6.0+cu124 |
| GPU | RTX 5060 Laptop (⚠️ sm_120, torch 2.6不兼容, 可能CPU回退) |
| WebUI端口 | 9874 |
| 界面语言 | 中文 |

### 4.2 启动

```bash
cd "D:\ai视频\角色设计_仓仓\GPT-SoVITS"
python webui.py zh_CN
```

浏览器打开 `http://localhost:9874`

### 4.3 关闭

关闭终端窗口，或 `Ctrl+C`

### 4.4 端口冲突

```bash
# 查占用
netstat -ano | findstr 9874
# 杀进程 (替换PID)
taskkill /F /PID <PID>
```

### 4.5 语音克隆流程

1. **准备音频**: 5秒以上干净语音 (WAV/MP3)，无背景噪音
2. **打开WebUI** → `http://localhost:9874`
3. **语音克隆标签页**: 上传音频 → 提取音色特征
4. **TTS标签页**: 输入文本 → 选择克隆的音色 → 生成

### 4.6 仓仓语音参考

| 场景 | 台词 | 备注 |
|------|------|------|
| 打招呼 | 啾姆啾姆~ 我是仓仓! | 标志性声音 |
| 开心 | 嘿嘿~ 草莓酸奶最好喝了! | 尾音上扬 |
| 生气 | 哼! 不许抢我的酸奶! | 鼓腮帮同步 |
| 害羞 | 啾姆... 别看啦... | 音量变小 |
| 日常 | 今天也要加油哦~ 啾姆! | 元气感 |

### 4.7 已知限制

- RTX 5060 对 CUDA 12.4 以下不兼容 → TTS推理可能自动CPU回退 (速度较慢)
- 如要GPU加速: 需升级torch到 2.7+cu128 nightly
- v2Pro模型缺失 (不影响基础功能)
- 日语/韩语/粤语依赖未安装 (仅中文可用)

### 4.8 故障排查

| 症状 | 原因 | 解决 |
|------|------|------|
| ModuleNotFoundError: opencc | 未安装 | `pip install opencc` |
| 端口9874被占用 | 上次未正常关闭 | 见4.4 |
| CUDA/sm_120 warning | GPU不兼容 | 忽略, CPU回退自动生效 |
| v2Pro模型warning | 模型缺失 | 忽略, 不影响基础TTS |

## 6. AI视频制作 (未来)

### 6.1 工作流

本项目的上游参考工作流 (来自 `D:\ai视频\CLAUDE.md` 和 `珠泪` 项目):

```
Phase 1: 脚本创作 → 闸门确认
Phase 1.5: 分镜图层 (角色圣经图+分镜图) → 闸门确认
Phase 2: 视频出片 (F2V+长时长+时间分段)
  Step 0: 全片锚点分析 (必执行)
  Step 1: Master Key Frame锁定
  Step 2: 镜头分组
  Step 3: 逐段出片
  Step 4: 后期最少化
```

### 6.2 出片信息包 (每段必须)

1. 成本(积分)
2. 参考图列表(含绝对路径)
3. 视频提示词(含锚点+转场)
4. 预期效果+衔接锁链
5. 即梦设置
6. 保存文件名

### 6.3 即梦成本参考 (VIP模型)

| 模式 | 实测消耗 |
|------|----------|
| I2V 图生视频 | ~11积分/秒 |
| F2V 首尾帧 | 11积分/秒 |
| 生图 (分镜) | 3积分/张 |

### 6.4 仓仓作为视频角色的注意事项

- 人形 vs Q版切换: 需要明确分镜标注形态
- 标志性元素锁定: 水母头/粉色/草莓酸奶/腮红 → 锚点
- 声音: GPT-SoVITS预录制 → 后期剪映对齐
- 动作库: 鼓腮帮(标志)/抱酸奶/蹦跳/挥手

## 7. 备份与迁移

### 7.1 最小备份集

必须备份:
- `角色设定_完整版.md` (角色核心设定)
- `角色设定.json` (结构化设定)
- `output/storyboards/` (13张圣经图)
- `docs/` (所有设计文档)

可选备份:
- `GPT-SoVITS/GPT_SoVITS/pretrained_models/` (3.4GB, 可重新下载)

### 7.2 迁移到新电脑

```bash
# 1. 复制角色数据
robocopy "D:\ai视频\角色设计_仓仓" "X:\新位置" /E /XD GPT-SoVITS

# 2. 在新电脑上重新搭建TTS
git clone --depth 1 https://github.com/RVC-Boss/GPT-SoVITS.git
cd GPT-SoVITS
pip install -r requirements.txt
# 复制 pretrained_models 或重新下载

# 3. 启动
python webui.py zh_CN
```
